# Руководство 1.1: Саморазвитие

## О руководстве

Это руководство содержит базовые практики и методы саморазвития, которые помогут вам выстроить фундамент для осознанной жизни.

## Для кого

- Начинающие практики саморазвития
- Те, кто хочет структурировать свой подход к развитию
- Участники программы MIM

## Результаты обучения

После изучения руководства вы сможете:

1. Управлять своим вниманием с помощью техники Pomodoro
2. Вести дневник внимания для отслеживания прогресса
3. Применять базовые практики осознанности
